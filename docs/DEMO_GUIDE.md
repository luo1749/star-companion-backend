# 🎯 星伴平台 - 演示指南

## 📋 演示准备

### 1. 启动顺序
1. 运行 `demo.bat` 启动所有服务
2. 或手动启动：
   - 后端：`cd backend && npm start`
   - 管理后台：`cd frontend\admin-web && npm run dev`
   - 小程序：微信开发者工具中打开

### 2. 访问地址
- **后端API**：http://localhost:3000
- **管理后台**：http://localhost:5173
- **小程序**：微信开发者工具中预览

## 🚀 快速演示流程（5分钟）

### 第1分钟：介绍项目
- 展示管理后台主界面
- 展示小程序界面
- 说明项目目标：特殊学生健康管理

### 第2分钟：展示核心功能
1. **数据监测**
   - 查看学生实时心率
   - 查看情绪状态
   - 查看设备状态

2. **预警系统**
   - 显示预警列表
   - 演示处理预警
   - 模拟发送预警

### 第3分钟：演示交互
1. **模拟心率数据**
   - 点击"模拟心率数据"按钮
   - 观察心率变化
   - 触发预警（心率>120）

2. **模拟预警**
   - 点击"模拟预警"按钮
   - 查看新增预警
   - 演示处理预警

### 第4分钟：展示多端协同
1. **管理后台**
   - 展示数据分析
   - 展示系统日志

2. **小程序端**
   - 展示家长视角
   - 演示紧急求助

### 第5分钟：技术亮点
1. **实时数据更新**（每10秒自动刷新）
2. **预警自动触发**
3. **多端数据同步**
4. **响应式界面设计**

## 🛠 故障排除

### 常见问题
1. **端口占用**：修改backend/server.js中的PORT
2. **无法访问**：检查防火墙设置
3. **数据不更新**：刷新页面或重启服务
